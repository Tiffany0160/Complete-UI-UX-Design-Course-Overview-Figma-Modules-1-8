# Complete-UI-UX-Design-Course-Overview-Figma-Modules-1-8


Project Overview

This project is a comprehensive visual guide covering the key concepts, tools, and workflows for learning UI/UX design using Figma. The infographic combines all eight modules of a beginner-to-advanced Figma course, providing an at-a-glance understanding of UI/UX principles, design processes, and practical applications.

It is designed for students, beginners, or aspiring UI/UX designers who want a structured overview of Figma and interactive design workflows.

Modules Included
Module 1: Introduction to Figma
UI/UX basics and importance
Creating an account
Exploring workspace, panels, layers, and menus
Module 2: Basic Tools and Shapes
Frames, shapes, text tools
Pen tool and vector shapes
Layers, grouping, alignment
Snapping, spacing, and grids
Module 3: Design Process and Audience Research
Understanding a design brief
User research and target audience
Paper wireframing
Translating paper wireframes to digital screens
Module 4: Wireframes & Prototypes
Low-fidelity vs high-fidelity wireframes
Creating interactive prototypes
Navigation flow and user experience mapping
Module 5: Design Resources & Styles
Free icons, fonts, and UI kits
Color theory and typography
Components and reusable styles in Figma
Module 6: Website & App Architecture
Structuring screens for web or mobile apps
Mapping user journey
Organizing pages, menus, and navigation
Module 7: Collaboration & Feedback
Real-time collaboration in Figma
Sharing projects and gathering feedback
Version control and comments
Module 8: Final Project
Developing a complete interactive prototype in Figma
Applying wireframes, interactive components, and design principles
Preparing for portfolio or presentation
Example projects: To-Do List App, Recipe App, Event Planner
Practical Applications


Each module includes hands-on exercises to reinforce learning, such as:
Creating frames, buttons, and input fields
Building low-fidelity and high-fidelity wireframes
Designing clickable prototypes
Collaborating with others and implementing feedback
